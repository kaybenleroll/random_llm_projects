# SILO RPG CAMPAIGN REFERENCE

Fast-launch reference for party setup and the eight starter campaigns.

## Party Bonds

| d6 | Bond |
| :--- | :--- |
| **1** | **The Work Crew:** You bonded during a dangerous cleanup assignment. |
| **2** | **The Block Watch:** You formed a neighborhood defense group. |
| **3** | **The Syndicate Debt:** You all owe the same dangerous boss. |
| **4** | **The Framed:** You were jailed for the same crime you did not commit. |
| **5** | **The Secret Society:** You share forbidden beliefs and meet in secret. |
| **6** | **The Relic Hunters:** You are an illegal crew built around contraband work. |

## Campaigns at a Glance

| Campaign | Theme | Location | Premise |
| :--- | :--- | :--- | :--- |
| **1. The Partition War** | Community friction, survival, housing | Level 92 | Two families are forced into one apartment after a collapse, while Nel Ardin tries to help from inside Supply. |
| **2. The Chit-Poor Harvest** | Hunger, gray market, desperation | Hydroponics, 60-80 | Crop loss leaves a whole block near starvation after Nel warns the party the ledgers are lying. |
| **3. The Flickering Dark** | Infrastructure decay, solidarity, resentment | Level 105 | Load shedding turns a hallway into a dark pressure cooker, and Nel buys the block a little time. |
| **4. The Down-Shift Transfer** | Bureaucracy, heartbreak, class divide | Levels 45-50 and below | The party's friend Nel Ardin is forced to leave for the Down Deep after marriage. |
| **5. The Rust in the Reclamation** | Bureaucracy, contamination, moral ambiguity | Mids, 80-85 | A toxic rust-bloom is poisoning families through the water supply. |
| **6. The Missing Porter** | Cults, hidden geography, forbidden belief | Levels 79-81 | A beloved Porter vanished with a critical Judicial message. |
| **7. The Lottery Swap** | IT espionage, paranoia, family desperation | Up Top / Upper Mids, 20-34 | A legitimate Lottery winner is erased from the official records. |
| **8. The Generator Hammer** | Industrial terror, class warfare | Down Deep, 130-144 | A coded SOS seems to come from beneath the Generator. |

## Quick Kits

### 1. The Partition War

* **Key Faces:** Jora Miller, Tavin Holt, **Nel Ardin**, Bee, Mett Saar
* **Clue Chain:** Missing heirloom -> child's hidey-hole -> collapse truth -> blocked requisition -> parts source
* **Escalation:** Tension rises -> hallway brawl -> deputy intervention -> forced relocation or arrest
* **Fail Forward:** The thief returns the item once violence becomes real

### 2. The Chit-Poor Harvest

* **Key Faces:** Foreman Dera Vos, Minn Ose, Cala Venn, Brikk, **Nel Ardin**
* **Clue Chain:** Yield shortfall -> skim -> processing problem -> textile favor debt -> hungry block pressure
* **Escalation:** Half-rations -> thefts -> child collapse -> names traded for food
* **Fail Forward:** Lost food can become leverage for broader relief

### 3. The Flickering Dark

* **Key Faces:** Rella Voss, Inspector Dorn, Tekk Mar, Yune, **Nel Ardin**
* **Clue Chain:** Dark hallway trouble -> power disparity -> industrial socket -> illegal splice -> inspection
* **Escalation:** Crime spikes -> pump fails -> stale-air sickness -> crackdown
* **Fail Forward:** Even a failed splice makes the PCs visible local heroes

### 4. The Down-Shift Transfer

* **Key Faces:** **Nel Ardin**, Rho Kell, Supervisor Pell, Hobb Fen
* **Clue Chain:** Missing manifest -> office access -> forged delay stamp -> hidden file drawer -> travel route
* **Escalation:** Deadline nears -> neighbors turn cold -> rival alerts Judicial -> dawn deadline
* **Fail Forward:** The party can still win a temporary extension by proving sabotage

### 5. The Rust in the Reclamation

* **Key Faces:** Deputy Mara Quill, Doctor Pell Rane, Kett Varo, Nix "Red-Hand" Sol
* **Clue Chain:** Red residue -> melted container -> Supply mismatch -> courier route -> hidden lab
* **Escalation:** More sickness -> Hydroponics scare -> quarantine panic -> Judicial sanitation team
* **Fail Forward:** A dying victim or observant Porter supplies the missed clue

### 6. The Missing Porter

* **Key Faces:** Elias Dane, Sister Avrel, Deputy Colm Werr, Bessa Pike
* **Clue Chain:** Blood and satchel -> porter schedule gap -> architectural mismatch -> vent grate -> Level 80.5
* **Escalation:** Family asks publicly -> Judicial notices -> cult relocates Elias -> execution begins
* **Fail Forward:** A runner reveals Elias used an impossible route

### 7. The Lottery Swap

* **Key Faces:** Lysa Trent, Inquisitor Sel Verin, Enid Marr, Doctor Hal Wen
* **Clue Chain:** Blood-test hardcopy -> burn order -> terminal log -> forged credentials -> family tie
* **Escalation:** Rumors spread -> records vanish -> Verin frames the PCs -> public hearing or silent arrest
* **Fail Forward:** A morgue attendant remembers the key order for a price

### 8. The Generator Hammer

* **Key Faces:** Shift-Boss Rourke, Iri Vale, Joss Trent, Old Merek
* **Clue Chain:** Flicker code -> miner's crew -> blocked route -> false checkpoint ledger -> vault noise
* **Escalation:** Pounding worsens -> Mechanical lockdown -> witnesses silenced -> vault crisis
* **Fail Forward:** Even a messy infiltration still gets them to the SOS

## Universal Running Notes

* **Best opening:** Start with a queue, work shift, or communal problem rather than a mission briefing.
* **Best twist:** Reveal that a personal problem is actually being shaped by paperwork, surveillance, or scarcity.
* **Best ending tone:** Let victories feel partial, costly, or temporary rather than clean.
